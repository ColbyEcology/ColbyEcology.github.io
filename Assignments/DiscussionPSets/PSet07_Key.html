The Lotka-Volterra competition equations (6a and 6b in your text) are described: \[ \frac{\mathrm{d}N_1}{\mathrm{d}t} = r_1N_1\left(\frac{K_1 - N_1 - \alpha _{12}N_2}{K_1}\right) \\ \frac{\mathrm{d}N_2}{\mathrm{d}t} = r_2N_2\left(\frac{K_2 - N_2 - \alpha _{21}N_1}{K_2}\right). \] This is similar to what youâve seen with mutualism and predator-prey, but with both of the interaction terms having negative signs. For these equations, first algebraically find the nullclines. Next, plot them all (not using phaseR), for the 4 different outcomes of competition; i.e., remake figure 8.4.

The nullclines are \(N_i^* = K_i - \alpha_{ij}N_j\).

comp_plot <- function(K1, K2, alpha12, alpha21) {
  plot(x = NA, type = "n", xlab = "N1", ylab = "N2", xlim = c(0, 1), ylim = c(0, 1), las = 1)
  segments(x0 = 0, y0 = 0, x1 = 0, y1 = 1, col = "red")
  segments(x0 = 0, y0 = 0, x1 = 1, y1 = 0, col = "blue")
  N1 <- seq(from = 0, to = 1, by = 0.1)
  N2_null <- K2 - alpha21*N1
  N1_null <- K1/alpha12 - 1/alpha12*N1
  lines(x = N1, y = N1_null, col = "red")
  lines(x = N1, y = N2_null, col = "blue")
}

comp <- function(t, y, parms) {
  with(as.list(c(y, parms)), {
  dN1 <- r1*N1*(K1 - N1 - alpha12*N2)/K1
  dN2 <- r2*N2*(K2 - N2 - alpha21*N1)/K2
  return(list(c(dN1, dN2)))
})
}
